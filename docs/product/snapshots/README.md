## Capture Standard

- Use real app screenshots from simulator or phone, not generated mockups.
- Prefer a recent iPhone simulator or a real iPhone screen.
- Keep personal or sensitive test text out of screenshots.
- Use demo or test data that shows the product clearly.
- Replace images when the UI changes meaningfully.
